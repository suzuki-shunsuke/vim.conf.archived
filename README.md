# vim.conf

hubが導入済みとする

```
$ git clone suzuki-shunsuke/vim.conf ~/gh/ss
$ ln -s ~/gh/ss/vim.conf/.vimrc ~/.vimrc
$ ln -s ~/gh/ss/vim.conf/.vim ~/.vim
$ mkdir ~/.vimundo
# install NeoBundle
$ curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh | sh
$ vi # プラグインインストール
```
